# Note on Erdős Problem 311

We let $\delta(N)$ be the quantity defined in the problem statement at
[Erdős Problem 311](https://www.erdosproblems.com/311).

This note proves a nontrivial upper bound for $\delta(N)$ using a result of Liu–Sawhney [arXiv:2404.07113](https://arxiv.org/abs/2404.07113v1).
